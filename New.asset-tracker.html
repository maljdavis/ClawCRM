⚡ One-time setup — deploy 3 Edge Functions to unlock full user management
After this, adding/removing branches and resetting passwords all happens right here.

Steps:
1. Go to supabase.com/dashboard → your project → Edge Functions → Deploy a new function
2. Deploy these 3 functions (code is in supabase/functions/ in your repo):
create-branch-user reset-branch-password delete-branch-user
3. Done — Add Branch, Edit (password reset), and Remove will all manage login accounts automatically.
