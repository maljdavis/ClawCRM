Branches / Locations

Manage branch access and users

⚡ One-time setup — deploy 3 Edge Functions to unlock full user management
After this, adding/removing branches and resetting passwords all happens right here.

Steps:
1. Go to supabase.com/dashboard → your project → Edge Functions → Deploy a new function
2. Deploy these 3 functions (code is in supabase/functions/ in your repo):
create-branch-user reset-branch-password delete-branch-user
3. Done — Add Branch, Edit (password reset), and Remove will all manage login accounts automatically.

Google Sheets Sync

Auto-sync assets from your Google Sheet or paste data manually

Setup (one time): In Google Sheets go to File → Share → Publish to web. Select your sheet tab, choose CSV, and click Publish. Paste the URL below.
The published link is a long unguessable URL — only people with the exact link can access it. It is not indexed by search engines.
How it works: Select your data in Google Sheets (including the header row), press Ctrl+C, then paste below. Existing records with the same serial/name are updated, not duplicated.
